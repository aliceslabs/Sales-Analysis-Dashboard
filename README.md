# Sales-Analysis-Dashboard
Personal Project to practice skills in SQL Data Cleaning &amp; Extraction, and Power BI Dashboard Creation. 

Used the AdventureWorks Sample Database: 
https://learn.microsoft.com/en-us/sql/samples/adventureworks-install-configure?view=sql-server-ver15&tabs=ssms

The business request for this data analyst project was an executive sales report for sales managers. Based on the Business request, the following user stories were defined to fulfill delivery and ensure that the acceptance criteria was maintained throughout the project.

| As a (role)  | I want (request / demand) | So that I (user value) | Acceptance Criteria | 
| ------------- | ------------- | ------------- | ------------- |
| Sales Manager | To get a dashboard overview of internet sales  | Can identify which customers and products sell the best  | A Power BI dashboard that updates data once a day  |
| Sales Representative  | A detailed overview of Internet Sales per Customers  | Can see which customers buy the most and who I can sell more to  | A Power BI dashboard that allows me to filter data for each customer  |
| Sales Representative  | A detailed overview of Internet Sales per Products  | Can see which Products sell the most  | A Power BI dashboard that allows me to filter data for each Product |
| Sales Manager  | A dashboard overview of internet sales  | Follow sales over time against budget  | A Power BI dashboard that contains graphs and KPIs comparing against budget.  |
